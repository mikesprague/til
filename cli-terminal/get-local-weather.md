# Get Local Weather from the Command Line

You can get weather information from command line using cURL and wttr.in:

```bash
# By Geolocation
curl -4 wttr.in

# By City
curl -4 wttr.in/Ithaca

# By 3 Letter Airport Code
curl wttr.in/ith

# Get Help/Instrctions
curl wttr.in/:help
```
